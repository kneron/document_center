# Kneron PLUS 3.2.0  

## Features  
N/A  

## Changes  
- Upgraded KL730 `kp_loader` / `kp_firmware` to SDK v2.0.0.  
    - Supports only BSP versions based on SDK v2.0.0 or later.  
    - Requires EVM/HDK upgrade.  
- Enhanced error handling for dynamically allocated memory in `pre/post_proc_config`.  
- Windows platform updates:  
    - Included `libusb` in the release package.  
    - Linked the standard library in the MinGW/MSYS environment.  

## Bug Fixes  
- Fixed `quantized_axis_stride` calculation when performing channel-wise quantization on input/output nodes.  
- Fixed a negative radix issue in the Python ONNX-to-NPU converter.  
- Corrected a data entry error in the documentation.  

## The following components are contained in Kneron PLUS:  
- KP API (C/Python)  
- Example code  
- ModelZoo example code  
- KDP2 firmware code  
- Pre-build firmware binary files  
- NEF files for demonstration purposes
